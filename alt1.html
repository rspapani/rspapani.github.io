MILL (MIPS Intermediary Logic Language): A language that serves as an intermediary between MIPS Assembly and C for more expressive yet traditionally intuitive low-level programming with a compiler written in Python. Allows one to abstract registers and stack frames withvariables, for/while loops, and function calls while still letting them be accessible in specific instances when desired. It is a statically-typed post-fix language with support for functional programming concepts such as currying and passing functions as values.
